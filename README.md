## 🚀 About The Project

This is my personal portfolio website, built to display my work, technologies I use, and ways to connect with me.  
It includes sections like:

- Hero / Introduction
- About Me
- Skills
- Projects
- Experience
- Contact form
- Social links

This project is fully responsive and modern, designed to look great on desktop and mobile.

---

## 🧠 Built With

The website is developed using:

- **React** – JavaScript library for building UI    
- **Tailwind CSS** – Utility-first CSS framework  
- **Vercel** – Deployment platform

## 🛠️ Features

✔ Fully responsive layout

✔ Fast performance and modern UI
